# jsWorking

- jsWorking is a markdown doc and ppt tool based on web.
- jsWorking是一款基于Web的用markdown编写html格式的doc和ppt的单文档工具,支持程序语法高亮，数学公式，结构图和统计图，可下载到本地使用，也可在线使用。

## 主要特性

- Markdown编写DOC
- Markdown编写PPT
- Latex方式编写公式
- 程序语法高亮显示
- 实时预览编写效果
- 本地打开与保存
- 导出HTML文件
- 自动暂存数据
- 画结构图
- 画统计图
